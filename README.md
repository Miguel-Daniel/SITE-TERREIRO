## Sobre o Projeto

Este projeto consiste no desenvolvimento de um site institucional para um terreiro de matriz africana, criado de forma independente como trabalho de autoria própria.

O site foi desenvolvido como parte de um Projeto Integrador acadêmico, reunindo conceitos de design, desenvolvimento web e valorização cultural. Após sua conclusão, será disponibilizado gratuitamente para a comunidade e para o terreiro que serviu de inspiração, como forma de contribuição e fortalecimento das tradições afro-brasileiras.

Todo o conteúdo, estrutura e implementação foram realizados de forma autoral. Como apoio no processo criativo e técnico, foram utilizadas ferramentas de inteligência artificial, como o ChatGPT, na geração de ideias e auxílio na construção de códigos, e o Gemini, na criação de imagens de exemplo utilizadas no projeto.
